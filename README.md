# Crypto Frontend

Este projeto foi desenvolvido para pesquisar o valor do Bitcoin em diferentes tipos de moedas utilizando React, ContextAPI, Hooks e ReactRouter.

A API você pode encontrar aqui https://github.com/renanvy/crypto_api

### Antes de rodar o projeto é necessário:
* Ter instalado NodeJS 15.0.1
* Ter instalado yarn 1.22.10 https://yarnpkg.com/

Obs: Você pode utilizar um gerador de versões como asdf https://github.com/asdf-vm/asdf para controlar as versões de cada linguagem

### Dependências
Instale as dependências do projeto rodando no terminal o comando: `yarn install`

### Testes
Para executar todos os testes rode o seguinte comando no terminal: `yarn test`

### Lint
Para verificar se o código está pradronizado com Eslint, rode o seguinte comando no terminal: `yarn lint`

### Servidor da API
Para iniciar o servidor da API instale e execute o projeto https://github.com/renanvy/crypto_api na sua máquina

### Iniciando a aplicação frontend
Para iniciar a aplicação rode o seguinte comando no terminal: `yarn start`
